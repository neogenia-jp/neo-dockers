# ユーザ管理

## 変数説明

`group_vars` に変数を定義してください。

  - `users`: 登録ユーザの一覧。ここに定義されているユーザアカウントがOSにログイン可能となります。
  - `del_users`: 削除済みユーザの一覧。ここに定義されているユーザはOSから削除されます。

## 新規ユーザの追加
